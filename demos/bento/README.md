# Bento

> Meal prep that ships itself

Meal prep and recipe app. Browse recipes, save favourites, build a grocery list, cook along with step timers. Built as a demo for the Yaver product videos — real working app with intentional bugs that the auto-fix loop repairs on camera.

---

## Stack

- **Mobile** (`apps/mobile`) — Expo RN (iOS + Android)
- **Backend** (`backend/`) — convex
- **Shared** (`packages/shared`) — cross-surface TS types + utils

Auth: Apple, Google, Email+password
Payments: none

## Quick start

```bash
npm install
./scripts/dev.sh    # runs every app in dev mode
./scripts/deploy.sh # builds + deploys every surface
```
