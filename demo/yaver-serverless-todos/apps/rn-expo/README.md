# Todo RN — Yaver Serverless

Expo Todo app backed by the Yaver Serverless Lite data API. It is separate from
`demo/mobile/todo-rn`, which stays local-only for feedback SDK videos.

```bash
cd demo/yaver-serverless-todos/apps/rn-expo
npm install
npm run ios
```

Set the backend URL, project slug, and `pp_` token in the app.
