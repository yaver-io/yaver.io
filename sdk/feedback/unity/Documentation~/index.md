# Yaver Feedback for Unity

Yaver brings self-hosted feedback, remote iteration, and agent-driven workflows to Unity projects.

Core capabilities:

- in-game feedback overlay
- auth and agent discovery
- screenshots, logs, crashes, and black-box capture
- vibing and feedback-to-fix workflows
- content refresh, scene reload, build, and relaunch paths

Suggested starting points:

- `YaverFeedback.Initialize(...)`
- `YaverContentRefreshHandler`
- `YaverFeedback.RunUnityTestsCoroutine(...)`
- `YaverFeedback.BuildConfiguredUnityCoroutine(...)`
- `YaverFeedback.RelaunchConfiguredUnityCoroutine(...)`

For the full package guide, see the repository README for this package.
