# Yaver — Downloads

## Desktop Agent

Install the Yaver agent on your development machine with the supported npm path:

```bash
npm install -g yaver-cli
```

## System Requirements

### Desktop Agent
- macOS 12+ / Windows 10+ / Ubuntu 20.04+
- 512 MB RAM
- Claude CLI installed (`npm install -g @anthropic-ai/claude-code`)
- Internet connection (for initial auth + peer discovery)

### Mobile App
- iOS 16+ / Android 12+
